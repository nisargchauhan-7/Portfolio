# 🚀 Personal Portfolio Website

A premium portfolio website inspired by modern digital agencies, built to showcase my skills, projects, and experience as a developer and designer.

## ✨ Features

* Modern agency-style UI/UX
* Fully responsive design
* Interactive GSAP animations
* Custom cursor and spotlight effects
* Animated project showcase
* Skills & technologies section
* Experience and education timeline
* Contact section
* Smooth scrolling navigation
* Premium hover interactions
* Floating background effects

## 🛠️ Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript (ES6)

### Libraries

* GSAP
* ScrollTrigger
